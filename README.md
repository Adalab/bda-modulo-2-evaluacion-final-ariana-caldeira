
DATA ANALYST - FINAL ASSESSMENT - MODULE 2
Ariana Caldeira


**Description**
This repository contains the SQL script developed for the final assessment of Module 2 of the Data Analytics bootcamp. The exercises were carried out using the Sakila database, a sample database that simulates a movie rental store.


**Repository Content**
- EF_modulo_2_DA_Ariana.sql : SQL script with exercises, queries, and comments.
- README.md : This file with the repository description.


**Requirements**
- SQL Server / MySQL


**Database Used**
The Sakila database includes tables such as:
- film
- actor
- customer
- rental
- category
- and other tables.

These tables are used to perform queries and data analysis in the context of a movie rental store.

ANALISTA DE DATOS - EVALUACIÓN FINAL - MÓDULO 2
Ariana Caldeira


**Descripción**
Este repositorio contiene el script SQL desarrollado para la evaluación final del Módulo 2 del bootcamp de Data Analytics. Los ejercicios fueron realizados utilizando la base de datos Sakila, una base de datos de ejemplo que simula una tienda de alquiler de películas.


**Contenido del Repositorio**
- EF_modulo_2_DA_Ariana.sql : Script SQL con ejercicios, queries y comentarios.
- README.md : Este archivo con la descripción del repositorio.


**Requisitos**
- SQL Server / MySQL


**Base de datos utilizada**
La base de datos Sakila incluye tablas como:
- film (películas)
- actor (actores)
- customer (clientes)
- rental (alquileres)
- category (categorías)
- entre otras tablas.

Estas tablas se utilizan para realizar consultas y análisis de datos en el contexto de una tienda de alquiler de películas.
